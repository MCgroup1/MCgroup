경고 5 : 강퇴 후 블랙리스트 1주.
1주 후 반성함을 확인 후에 블랙리스트 해제.

영구강퇴 : 강퇴 후 영구 블랙리스트 등재.
방장이나 부방장에게 지필 사과문 작성하면 블랙리스트 해제.

이보다 더 심한 경우엔 강퇴처리됨과 동시에 포스트에 박제되고, 해킹(MC ㅎㅇㅌ) 을 당하시던 말던 저희는 책임지지 않습니다.

싸우거나 무개념 적발 시,
관리자만 일시적으로 반말, 욕설 등 모든 규칙이 허용되며, 위의 내용처럼 저희는 책임지지 않습니다.
